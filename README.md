# Ikuu VPN - Minimal Version

A minimal Android VPN application with basic functionality.

## Features
- Simple UI with Jetpack Compose
- ikuu API integration
- Subscription management
- Clash Meta core support (optional)

## Architecture
- app: Main application module
- common: Shared utilities
- ikuu-api: ikuu VPN API integration
- clash-core: Clash Meta core (optional)

## Build Status
- [x] GitHub Actions: Working on automated build
- [x] Local Build: Automated scripts ready

## Usage
1. Clone the repository
2. Run `setup_and_build.bat` (Windows) or `setup_and_build.sh` (Linux/macOS)
3. Find the APK in `app/build/outputs/apk/debug/`
